Pengaturan API

Bagaimana cara mendapatkan API Key?
  1. Buka Google AI Studio.
  2. Klik tombol "Get API key" (biasanya di pojok kiri atas).
  3. Pilih atau buat Proyek Google Cloud jika diminta.
  4. Klik "Create API key" untuk membuat kunci baru.
  5. Salin (copy) kunci yang baru dibuat dan tempel (paste) di kolom atas.
